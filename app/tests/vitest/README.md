# Vitest tests

## Running tests

- `npm run vitest` (single run)
- `npm run vitest--watch` (watch mode)

## File structure

- Keep tests colocated with the source tree structure under `tests/vitest`.
- Name files with the `.test.ts` suffix.

## Helpers and fixtures

- Place shared test helpers under `tests/vitest/helpers`.
- Place fixtures under `tests/vitest/fixtures`.
